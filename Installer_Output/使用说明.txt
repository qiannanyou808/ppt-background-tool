=====================================
PPT背景替换工具 - 用户使用说明
=====================================

【安装步骤】
-----------

1. 双击"PPT背景替换工具_v1.2.0_安装程序.exe"

2. 选择语言（English）→ 点击 OK

3. 阅读许可协议 → 勾选 "I accept..." → Next

4. 选择安装位置（默认即可）→ Next

5. 选择是否创建桌面图标 → Next

6. 点击 Install 开始安装

7. 等待安装完成 → Finish

8. 完成！可以从开始菜单或桌面运行


【使用方法】
-----------

1. 运行"PPT背景替换工具"

2. 点击"选择PPT文件"，选择要处理的PPT（可多选）

3. 点击"选择背景图片"，选择新的背景图

4. 点击"开始替换背景"

5. 等待处理完成

6. 在原文件目录找到"xxx_新背景.pptx"


【注意事项】
-----------

✓ 仅支持.pptx格式（不支持旧版.ppt）
✓ 原文件不会被修改，会生成新文件
✓ 支持中文路径和文件名
✓ 背景图片建议使用16:9或4:3比例


【系统要求】
-----------

✓ Windows 7/8/10/11（64位）
✓ 不需要安装Python
✓ 约200MB硬盘空间


【常见问题】
-----------

Q: 双击没反应？
A: 可能缺少运行库，请安装：
   https://aka.ms/vs/17/release/vc_redist.x64.exe

Q: 杀毒软件报警？
A: 这是误报，添加到白名单即可

Q: 如何卸载？
A: 控制面板 → 程序和功能 → 找到"PPT背景替换工具" → 卸载
