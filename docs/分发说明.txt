=====================================
PPT背景替换工具 - 分发指南
=====================================

版本：v1.2.0


【推荐分发方案】
===============

方案1：单文件EXE（最简单）
-------------------------
优点：只有一个文件，方便传输
缺点：启动稍慢，可能被杀毒软件误报

步骤：
1. 运行 build_exe_spec.bat 打包
2. 获取文件：dist\PPT背景替换工具.exe
3. 发给用户：直接发送EXE
4. 用户使用：双击运行

文件大小：约 80-150 MB


方案2：文件夹版（推荐）
-----------------------
优点：启动快，误报少
缺点：需要压缩整个文件夹

步骤：
1. 运行 build_exe_folder.bat 打包
2. 获取文件夹：dist\PPT背景替换工具\
3. 压缩成ZIP：PPT背景替换工具.zip
4. 发给用户：用户解压后运行
5. 用户使用：解压 → 进入文件夹 → 双击 PPT背景替换工具.exe

文件大小：约 100-200 MB（压缩后约 40-80 MB）


方案3：完整安装包（最专业）
---------------------------
包含内容：
  - EXE文件或文件夹
  - README使用说明
  - Visual C++ Redistributable安装包
  - 示例PPT和背景图片

步骤：
1. 打包程序（方案1或2）
2. 下载 vc_redist.x64.exe
3. 编写简单的使用说明.txt
4. 准备示例文件
5. 全部打包成ZIP
6. 发给用户

这样用户收到后什么都不用担心，开箱即用！


【用户系统要求】
===============

✓ Windows 7/8/10/11（64位）
✓ 约 200 MB 硬盘空间
✗ 不支持 Mac / Linux（需要在对应系统重新打包）


【常见问题处理】
===============

问题1：双击EXE没反应
--------------------
可能原因：
  - 杀毒软件拦截
  - 缺少 Visual C++ Redistributable
  - 被Windows SmartScreen阻止

解决方法：
  1. 添加到杀毒软件白名单
  2. 安装 vc_redist.x64.exe
  3. 右键 → 属性 → 解除锁定
  4. 或点击"详细信息" → "仍要运行"


问题2：提示缺少DLL文件
----------------------
原因：缺少 Visual C++ Redistributable

解决方法：
  安装：https://aka.ms/vs/17/release/vc_redist.x64.exe


问题3：杀毒软件报警
------------------
这是正常现象！PyInstaller打包的程序常被误报。

解决方法：
  1. 添加到白名单
  2. 或暂时关闭杀毒软件
  3. 或使用代码签名（需购买证书）


问题4：中文乱码
--------------
极少数系统可能出现。

解决方法：
  系统设置 → 区域 → 管理 → 更改系统区域设置
  → 勾选"Beta: 使用 Unicode UTF-8"

【创建使用说明示例】
===================

PPT背景替换工具 - 使用说明

1. 双击"PPT背景替换工具.exe"启动程序
2. 点击"选择PPT文件"，选择要处理的PPT
3. 点击"选择背景图片"，选择新的背景图
4. 点击"开始替换背景"
5. 等待处理完成
6. 在原文件目录找到"xxx_新背景.pptx"

注意事项：
- 仅支持 .pptx 格式
- 原文件不会被修改
- 支持批量处理多个PPT

如有问题，请联系技术支持。


【高级：代码签名】
=================

如果需要避免杀毒软件误报，可以购买代码签名证书。

步骤：
1. 购买证书（约 $100-300/年）
2. 使用 signtool 签名EXE
3. 分发签名后的EXE

这样Windows会显示"已验证的发布者"，更专业。


【总结】
=======

最简单：方案1 - 单文件EXE
最推荐：方案2 - 文件夹版
最专业：方案3 - 完整安装包

根据你的分发对象和场景选择合适的方案！
